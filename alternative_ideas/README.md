## Stochastic Search
Our priority function gives us much more information than we're actually using. We always choose the highest priority item. In doing so, we treat priority as a binary. The aim of stochastic search is to adapt our solve function to use all the information the priority function provides by using differences in priorities between bins to determine how often we should pick one over the other. Making these random choices using a high-performing priority function could help us perform a brute-force search of sorts that tends to explore higher quality partitions. Unfortunately, this hasn't worked out so far. 
## Template Funsearch 
Following Ageron et al. (2022), the WS+_spec1.py is a first implementation of the fun search algorithm searching for $b$-WS template that optimizes the bounds it provides on WS$(n)$ for a certian $n$ value. It has some hard-coded parameters and I'm not sure the implementation is entirely correct. In any case, it has not produced noteworthy results.
